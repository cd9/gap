
# GAP: Generative Art Playground

This is an experiment with generating art using the Pillow library for Python.
The idea is to build up a set of classes that paint onto the canvas in wacky ways.

# Features
- [x] Argument parsing
- [ ] Diamond paint tool
- [ ] Border paint tool
- [ ] Star paint tool
